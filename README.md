# 🎬 Netflix Content & Customer Analytics Dashboard (Power BI)

## 📊 Project Overview

This project presents an interactive **Power BI dashboard** analyzing Netflix data to uncover insights about **content distribution, customer behavior, subscription trends, and global performance**.

The analysis is based on two datasets containing Netflix content details and customer viewing behavior.

---

## 🎯 Project Objectives

- Analyze Netflix **Movies vs TV Shows distribution**
- Understand **customer demographics and viewing patterns**
- Evaluate **subscription plan performance**
- Identify **top-performing content**
- Explore **country-wise content and engagement trends**

---

## 🛠 Tools & Technologies Used

- 📊 Power BI  
- 🧮 DAX (Data Analysis Expressions)  
- 🔄 Power Query (Data Cleaning & Transformation)  
- 📁 Excel Dataset  

---

## 📂 Dataset Used

### 🔹 1. Netflix Content Dataset  
📁 `netflix_dataset_20000.xlsx`

Contains:
- Show ID  
- Title  
- Director  
- Genre  
- Country  
- Release Year  
- Rating  
- Duration  
- Type (Movie / TV Show)  

---

### 🔹 2. Customer Ratings Dataset  
📁 `netflix_customer_ratings_20000.xlsx`

Contains:
- Customer ID  
- Show ID  
- Customer Name  
- Age  
- Gender  
- Country  
- Subscription Type  
- Watch Time (Minutes)  
- Rating  

---

## 🔗 Data Modeling

  - Content data  
  - Customer behavior

## 📊 Key Insights

- 🎬 Movies dominate Netflix content compared to TV Shows  
- ⭐ Majority of content is rated **TV-MA and TV-14**  
- 🌍 United States leads in content production  
- 📈 Significant growth in content after 2016  
- 💎 Premium users show higher engagement and watch time
